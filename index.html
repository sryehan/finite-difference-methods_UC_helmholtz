🔥 HEAT CONDUCTION

1D Steady & transient analysis

01
STEADY · 1D · VARIABLE λ

Mixed boundary conditions & matrix assembly

Solve −d/dx[λ(x) dT/dx] = 0 for x∈[0,0.4] with λ(x)=0.5+2x (W/°C·m).

λ(x) = 0.5 + 2.0x
BC: T(0)=200°C, q(L)=10 W/m² (Neumann)

The linear system is built using central differences for interior nodes. Neumann condition discretized via backward differences. The resulting temperature distribution shows strong influence of variable conductivity.

📄 MATLAB · FD assembly
% Domain discretization
L = 0.4; npts = 5; dx = L/(npts-1);
x = linspace(0,L,npts); lambda = 0.5 + 2.0*x;
A = zeros(npts); B = zeros(npts,1);
A(1,1) = 1; B(1) = 200;
A(end,end-1:end) = -lambda(end)*[-1/dx, 1/dx]; B(end) = 10;
for i = 2:npts-1
  A(i,i-1:i+1) = [-1, 2, -1]*lambda(i)/dx^2;
end
T = A\B;  % temperature field
02
TRANSIENT · EXPLICIT EULER

CFL condition · Right boundary temperature evolution

Transient heat diffusion: ∂T/∂t = α ∂²T/∂x² with α = λ/(ρ·Cp). Neumann flux = 100 W/m² at right end, Dirichlet left T=20 °C. The explicit scheme requires dt ≤ 0.5·dx²/α to ensure stability.

α = 2 / (200·1000) = 1e-5 m²/s
dt = 0.4·dx²/α (conservative CFL)

Rightmost temperature increases over time due to applied heat flux, reaching quasi-steady regime after ~200 time steps.

⏱️ Time marching (explicit FD)
% stability bound
alfa = lambda/(rho*Cp); dx = 0.01;
dt = 0.4*dx^2/alfa; nsteps = 200;
T = T0*ones(npts,1);
for it = 1:nsteps
  Tnew = T;
  for j = 2:npts-1
    Tnew(j) = T(j) + dt*alfa*(T(j-1)-2*T(j)+T(j+1))/dx^2;
  end
  Tnew(end) = Tnew(end-1) + flux_right*dx/(-lambda);
  T = Tnew;
end
🌡️ 2D CONDUCTION

Steady-state · domain 3.5×2.0 m

03
LAPLACE 2D · MIXED BC

Temperature map & heat flux vectors

Laplace eqn ΔT = 0 with λ = 0.5 W/°C·m. Left (20 °C) and top (10 °C) Dirichlet, right & bottom adiabatic (q=0). After solving A·T = b, heat flux is recovered via qx = -λ ∂T/∂x (central differences).

∂²T/∂x² + ∂²T/∂y² = 0

Resulting isothermal contours show gradient towards warm left boundary. The flux field (quiver plot) indicates heat flowing from hot regions to cold top/left edges.

Fig. 1 — Temperature field (°C) with hot/cold gradient
🧮 2D FD assembly (Laplace)
% Build sparse matrix for nx×ny grid
for i = 1:nx, for j = 1:ny
  idx = (i-1)*ny+j;
  if i==1, A(idx,idx)=1; B(idx)=20;      % left Dirichlet
  elseif j==ny, A(idx,idx)=1; B(idx)=10;  % top Dirichlet
  elseif i==nx, % Neumann right
    A(idx,[idx-ny, idx]) = -lambda*[-1/dx, 1/dx];
  elseif j==1, % Neumann bottom
    A(idx,[idx, idx+1]) = -lambda*[-1/dy, 1/dy];
  else % interior: 5-point stencil
    A(idx,idx-ny) = -lambda/dx^2; A(idx,idx+ny) = -lambda/dx^2;
    A(idx,idx-1) = -lambda/dy^2; A(idx,idx+1) = -lambda/dy^2;
    A(idx,idx) = 2*lambda/dx^2 + 2*lambda/dy^2;
  end
end; end
T = A\B;
🎵 ACOUSTICS · FINITE DIFFERENCES

Helmholtz equation — 1D & 2D frequency domain

04
1D HELMHOLTZ

Analytical vs. FD complex pressure

d²P/dx² + K²P = 0, K = ω/c, c=343 m/s, f=1000 Hz. Dirichlet P=1 Pa at x=0, rigid termination (v=0) at x=L. Complex pressure field computed and compared with closed-form P_exact = cos(K(L-x))/cos(KL).

Neumann BC → ∂P/∂x = 0 (v=0)

Excellent agreement between numerical and analytical real/imag parts for moderate grid resolution.

🎛️ 1D Helmholtz matrix
K = 2*pi*freq/c;
k1 = 1/dx^2; k2 = -2/dx^2 + K^2;
for i = 2:n-1, A(i,i-1:i+1) = [k1, k2, k1]; end
A(1,1) = 1; B(1) = 1;
% rigid wall
A(end,end-1:end) = [-1/dx, 1/dx];
P = A\B;  % complex pressure vector
