01 — Heat Conduction

1D Steady-State Heat Conduction

01
Steady · 1D

Variable Conductivity, Mixed Boundary Conditions

Solve the steady-state heat conduction ODE over x ∈ [0, 0.5] with variable thermal conductivity:

−d/dx [ λ(x) · dT/dx ] = q(x) λ(x) = 0.5 + 2.0x (W/°C·m)

Boundary Conditions:

BoundaryTypeValue
x = 0DirichletT = 200 °C
x = LNeumannq = 10 W/m²

The FD system is assembled by looping through nodes. Dirichlet imposed directly; Neumann via backward differences; internal nodes via standard second-order scheme.

MATLAB
% Domain & conductivity
L = 0.4; npts = 5;
dx = L/(npts-1);
x = 0:dx:dx*(npts-1);
lambda = 0.5 + 2.0*x;

A = zeros(npts); B = zeros(npts,1);

% Dirichlet: left
A(1,1)=1; B(1)=temp_left;

% Neumann: right (backward diff)
A(end,end-1:end) = ...
  -lambda(end)*[-1/dx 1/dx];
B(end) = flux_right;

% Internal nodes
for ii = 2:npts-1
  A(ii,ii-1:ii+1) = ...
    -[1 -2 1]/dx^2*lambda(ii);
end
T = A\B;
02
Transient · 1D

Explicit Time-Marching with CFL Stability

Parabolic PDE governing transient heat diffusion, solved with explicit forward Euler in time and second-order central differences in space:

ρCp · ∂T/∂t = λ · ∂²T/∂x² Stability: dt ≤ (1/2) · dx² / α where α = λ / (ρ · Cp)

Parameters: λ = 2 W/°C·m, ρ = 200 kg/m³, Cp = 1000 J/kg·°C. Initial condition T₀ = 20 °C throughout. Neumann BC at right: flux = 100 W/m².

Temperature is marched through 200 time steps, tracking evolution at the rightmost node.

MATLAB — Time Loop
% CFL-stable time step
alfa = lambda/rho/Cp;
dt = (1/2.5)*dx^2/alfa;

Ti0 = T0;

for itime = 1:nsteps
  for ipos = 2:npts-1
    Ti1(ipos) = Ti0(ipos) + dt*alfa*...
      (Ti0(ipos-1)-2*Ti0(ipos)+...
       Ti0(ipos+1))/dx^2;
  end
  Ti1(1) = temp_left;
  Ti1(end) = flux_right*dx/...
    (-lambda)+Ti1(end-1);
  Ti0 = Ti1;
end
02 — Heat Conduction

2D Steady-State Heat Conduction

03
Steady · 2D

3.5 × 2.0 m Domain — Temperature Distribution & Heat Flux

The 2D Laplace equation is solved over a rectangular domain with mixed boundary conditions. λ = 0.5 W/°C·m.

∂²T/∂x² + ∂²T/∂y² = 0
BoundaryTypeValue
Left (x=0)DirichletT = 20 °C
Top (y=Ly)DirichletT = 10 °C
Right (x=Lx)Neumannq = 0 W/m²
Bottom (y=0)Neumannq = 0 W/m²

Heat fluxes qx and qy are computed post-solution using central differences at internal nodes.

Fig. 1 — Temperature distribution (°C), hotmap colorscale
MATLAB — 2D Assembly Loop
for ii = 1:nptx
  for ij = 1:npty
    center = (ii-1)*npty + ij;
    left=center-npty; right=center+npty;
    up=center+1;   down=center-1;

    if     x(center)==0    % Left: Dirichlet
      A(center,center)=1; B(center)=20;
    elseif y(center)==Ly   % Top: Dirichlet
      A(center,center)=1; B(center)=10;
    elseif x(center)==Lx   % Right: Neumann
      A(center,[left center])=-lambda*[-1 1]/dx;
    elseif y(center)==0    % Bottom: Neumann
      A(center,[center up])=-lambda*[-1 1]/dy;
    else                    % Interior: PDE
      A(center,[left center right])=...
        -lambda*[1 -2 1]/dx^2;
      A(center,[down center up])=...
        A(center,[down center up])...
        -lambda*[1 -2 1]/dy^2;
    end
  end
end
T = A\B;
03 — Acoustics

Helmholtz Equation — Finite Differences

04
Harmonic · 1D

1D Acoustic Pressure — Comparison with Analytical Solution

The frequency-domain Helmholtz equation governs acoustic pressure:

d²P/dx² + K²P = 0 K = ω/c = 2πf / 343 (rad/m)
BoundaryTypeValue
x = 0DirichletP = 1 Pa
x = LNeumannv = 0 m/s

The Neumann BC translates to −(1/iωρ)·∂P/∂x = 0, applied via backward differences. The complex pressure field is compared with the exact closed-form solution.

MATLAB — Helmholtz 1D
% Wavenumber
c=343; freq=1000;
K = 2*pi*freq/c;

% FD coefficients
k1 = 1/dx^2;
k2 = -2/dx^2 + K^2;
k3 = 1/dx^2;

for ii = 2:npts-1
  A(ii,ii-1:ii+1) = [k1 k2 k3];
end

% Neumann at right
A(end,end-1:end) = ...
  -1/1i/rho/w*[-1/dx 1/dx];

P = A\B;

2D Acoustic Pressure — 3.5 × 2.0 m Domain

The 2D Helmholtz equation is solved on a 66 × 41 structured point grid:

∂²P/∂x² + ∂²P/∂y² + K²P = 0 f = 1000 Hz, c = 343 m/s
BoundaryTypeValue
LeftDirichletP = 1 Pa
TopDirichletP = 1 Pa
RightNeumannv = 0 m/s
BottomNeumannv = 0 m/s

The complex pressure field is plotted as real, imaginary, and absolute values using trisurf with jet colormap.

MATLAB — 2D Assembly
for ii=1:nptx
 for ij=1:npty
  center=(ii-1)*npty+ij;
  if     x(center)==0
    A(center,center)=1; B(center)=1;
  elseif y(center)==Ly
    A(center,center)=1; B(center)=1;
  elseif x(center)==Lx
    A(center,[left center])=[-1 1]/dx;
  elseif y(center)==0
    A(center,[center up])=[-1 1]/dy;
  else
    A(center,[left center right])=...
      [1 -2 1]/dx^2;
    A(center,[down center up])=...
      A(center,[down center up])...
      +[1 -2 1]/dy^2;
    A(center,center)+=K^2;
  end
 end
end
P = A\B;
